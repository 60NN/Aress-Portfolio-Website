What's Inside

Features.

01
Magnetic Custom Cursor
A two-layer cursor system — a sharp dot that follows instantly, and a lagging ring that smoothly trails behind. Expands on hover over interactive elements.
requestAnimationFrame CSS Transform
02
Hero Typewriter Effect
The name animates character by character on load, with the first letter highlighted in accent color. A subtle touch that creates an immediate sense of craft.
setTimeout DOM Manipulation
03
Scroll-Triggered Reveals
Every section fades and rises into view as the user scrolls, using the IntersectionObserver API for buttery-smooth performance.
IntersectionObserver CSS Transitions
04
Project Hover Previews
Hovering over a project row reveals a floating preview card that follows the mouse — showing the project's emoji and background color in real time.
mousemove Fixed Positioning
05
Infinite Ticker Band
A continuous scrolling ticker in the brand's accent yellow, listing services in both English and Arabic — a nod to cultural identity.
CSS Animation Bilingual
06
Fully Responsive
From wide desktop to mobile, the layout adapts gracefully — collapsing navigation, reordering grids, and maintaining visual quality at every breakpoint.
CSS Grid Media Queries

How It Was Built

Process.
